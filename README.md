Bu proje, yüz tanıma teknolojisi kullanarak bir veri tabanında arama yapma ve eşleşen kişileri belirleme amacını taşımaktadır. Proje, bir kişinin görseli aratıldığında, kendi oluşturduğum veri tabanları arasında bu kişiyle eşleşen özellikler varsa, kişi adı ve soyadını eşleştiren bir sistem geliştirmektedir.
Python - OpenCV - dlib - deepface - retinaface kullanıldı
